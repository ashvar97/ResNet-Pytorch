## Getting Started
### Prerequisites
- **Python 3.x**
- **PyTorch**
- Install dependencies using:
  ```bash
  pip install -r requirements.txt
  ```

## Usage
### Training the Model
Run the training script:
```bash
python train.py
```

### Evaluating the Model
Run the test script:
```bash
python test.py
```
